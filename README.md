# 🌍 AI Language Translator

A simple, fast, and interactive Machine Translation web application built with Python and Streamlit.

### Features
* Translates text instantly between multiple languages.
* Powered by Google Translate's engine (via `deep-translator`).
* Built entirely on a mobile device!

### Tech Stack
* **Frontend/UI:** Streamlit
* **NLP/Translation:** deep-translator
